Requerimentos:

Primeiro você precisa saber se você está dentro das dependências mínimas
- Slackware64 Current
- Pacote de Áudio instalado

Como instalar?
Descompacte o arquivo com o comando "tar xvf cinnamon-gtk3-install.tar.xz"
Entre na pasta "install-gtk3" que acabou de ser criada
Então execute o script com o comando "sudo sh cinnamon-gtk3-install.sh"

Como compilar?
Descompacte o arquivo com o comando "tar xvf cinnamon-gtk3-source.tar.xz"
Entre na pasta "source-gtk3" que acabou de ser criada
Então execute o script com o comando "sudo sh build-cinnamon.sh"

How to install?
Unzip the file with the command "tar xvf cinnamon-gtk3-install.tar.xz"
Enter the "install-gtk3" folder you just created
Then run the script with the command "sudo sh cinnamon-gtk3-install.sh"

How to build?
Unzip the file with the command "tar xvf cinnamon-gtk3-source.tar.xz"
Enter the "source-gtk3" folder you just created
Then run the script with the command "sudo sh build-cinnamon.sh"
